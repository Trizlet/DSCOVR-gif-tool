# DSCOVR-gif-tool

A python tool to create gifs from imagery gathered by the EPIC instrument onboard the DSCOVR spacecraft.

Use -t/--type ['n', 'e', 'a','c'] to select natural, enhanced, aerosol, or cloud imagery.
